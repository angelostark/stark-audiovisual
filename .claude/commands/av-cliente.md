Ative o agente AV Monitor e execute o comando *por-cliente com o argumento fornecido pelo usuário.

Se o usuário digitou `/av-cliente Cadu`, o argumento é "Cadu".
Se nenhum argumento foi fornecido, pergunte: "Qual cliente você quer analisar?"

Instruções:
1. Carregue o agente em: squads/audiovisual-squad/agents/av-monitor.md
2. Carregue a task em: squads/audiovisual-squad/tasks/av-por-cliente.md
3. Use o ClickUp MCP: clickup_filter_tasks com list_ids: ["901324888130"] e subtasks: true
4. Filtre localmente as tarefas onde o título contém o nome do cliente (case insensitive)
5. Execute a task av-por-cliente seguindo exatamente as instruções do arquivo
6. Exiba o status completo de produção do cliente com tabela por semana

Argumento do usuário: $ARGUMENTS
